    HTML

    HTML (HyperText Markup Language) is used to give content to a web page and instructs web browsers on how to structure that content.

    HTML Element

    An HTML element is composed of three sections:

    • Opening tag
    • content
    • Closing tag
    Section Name Description
    <tag> Opening tag The opening tag of an HTML element, it can contain multiple attributes specific to said element.
    Some elements only include this tag.
    Hello World! Content The actual content that is render in the web page.
    </tag> Closing tag The closing tag of the html element. It is the same as the opening tag, but includes a foward slash (/) before the tag name.

    Basic html tags

    Some of the most common html tags are the ones capable of render text. Some examples include:

    Tag Name Description
    <h1> Heading text A text header. It goes from largest to smallest from <h1> to <h6>
    <p> Paragraph The paragraph element contains and displays a bloc of text.
    <ol> Ordered List The ordered list element creates a list of items in sequential order. Each list item appears numbered by default.
    <ul> Unordered List This element is used to create a list of items in no particular order. Each individual list item will have a bullet point by default.
    <li> List item This element creates list items inside the ordered and unordered list elements.
